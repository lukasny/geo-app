# Bulk Editing UI Implementation Plan

**Goal:** Select N audited products in a grid, apply meta title and alt-text templates via Shopify mutations, keep the local cache and audit results honest, and bound the webhook-triggered llms.txt regeneration cost.

**Architecture:** Extract an AI-free write layer from audit-engine (product-mutations.server.ts), build bulk-edit.server.ts on top of it, render an audit-style IndexTable with selection at /app/bulk-edit (Growth+ via bulkOptimization), and coalesce webhook llms.txt regenerations per store (llms-regen-queue.server.ts).

**Tech Stack:** Remix, TypeScript strict, Polaris v12 (first selectable IndexTable in the repo), Prisma, admin.graphql. Verification per task: `npx tsc --noEmit`; final: `npm run build` + multi-agent review.

**Spec:** `docs/superpowers/specs/2026-06-12-bulk-edit-design.md`

## Task 3: bulk-edit.server.ts

**Files:** Create `app/services/bulk-edit.server.ts`

- [ ] `renderTemplate(template, vars)` with {title} {vendor} {type} {handle} {price} {shop}, null-safe, whitespace-collapsed, trimmed.
